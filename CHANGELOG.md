# 변경 이력 (Changelog)

## [1.0.0] - 2025-12-11

### ✨ 주요 기능 추가

#### 초기 개발
- React + TypeScript 프론트엔드 구현
- Express 백엔드 서버 구현
- PDF 업로드 기능 (드래그 앤 드롭)
- 기본 PDF 파싱 로직 (정규표현식 기반)
- Excel 생성 기능 (4개 시트)

#### AI 통합
- Google Gemini 2.5 Flash API 통합
- PDF를 base64로 직접 전달하는 방식
- 재시도 로직 구현 (최대 3회)
- 폴백 시스템 (AI 실패 시 기존 파서 사용)

#### Excel 출력 개선
- 단일 시트로 통합 (사용자 요청)
- "현재 유효한 권리 요약" 형식으로 변경
- 섹션별 헤더와 테이블 구성
- 자동 합계 계산
- 페이지 설정 (한 페이지에 맞추기)

#### UI/UX 개선
- 다크 모드 디자인
- 글래스모피즘 효과
- 부드러운 애니메이션 (Framer Motion)
- 반응형 레이아웃
- 탭 기반 결과 탐색
- 실시간 진행률 표시

#### 보안 강화
- 환경변수로 API 키 관리 (dotenv)
- .env 파일 Git 제외
- .env.example 파일 추가
- 파일 자동 삭제 기능

#### 레이아웃 조정
- 메인 컨테이너 너비 확대 (max-w-4xl → max-w-6xl)
- 순위번호 컬럼 너비 증가 (min-w-[120px])
- 상태 컬럼 너비 증가 (min-w-[100px])
